Connect with Your Favorite Tool
You can also connect using tools like Cloude, MCP Inspector, or any other compatible client.
Just ensure you set the server URL to

https://your.app.url.databricksapps.com/api/mcp/

and use the Streamable HTTP Transport with the Authentication header.
Example Connection (Python) 

from databricks.sdk import WorkspaceClient
from mcp.client.streamable_http import streamablehttp_client as connect
from mcp import ClientSession

client = WorkspaceClient()

async def main():
    # Connect to a streamable HTTP server

    headers = client.config.authenticate()
    app_url = "https://your.app.url.databricksapps.com/api/mcp/"
    async with connect(app_url, headers=headers) as (
        read_stream,
        write_stream,
        _,
    ):
        # Create a session using the client streams
        async with ClientSession(read_stream, write_stream) as session:
            # Initialize the connection
            await session.initialize()
            # Call a tool
            tool_result = await session.call_tool("echo", {"message": "hello"})
